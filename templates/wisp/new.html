Cloud Wisps

Wisps are configuration templates used by Clouds and Appliances to start Instances. Wisps send SSH Keys and Post Creation Scripts to instances using Callback URLs accessed by the pool's appliances after they receive payments. Add a Custom Callback URL to your wisp if you want more control over your instance starts.

Note: Using a custom callback URL will cause the instance to ignore the image, ssh key and post creation script defined here.
